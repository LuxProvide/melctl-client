# Version

Return MelCtl client and server versions.

## Usage

```shell
melctl version
```

## Example return

Table:

```
serverVersion    serverVersionName    clientVersion    info
---------------  -------------------  ---------------  ------
5.0.1            Pedantic Santa       5.0.0            OK
```

YAML:

```yaml
clientVersion: 5.0.0
info: OK
serverVersion: 5.0.1
serverVersionName: Pedantic Santa
```
